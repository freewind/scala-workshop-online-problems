读取一行

scala.io.Source.stdin.readLine

读取多行（全部）

scala.io.Source.stdin.readLines

输出一行

println("content")

完整示例：读到什么显示什么

import scala.io.Source.stdin

object App {
  def main(args: Array[String]) {
    stdin.getLines foreach { line =>
      println(line)
    }
  }
}
通常你可以copy上面这段代码到作为模板

注意：输出内容前后的空白不影响结果对比
